## рџ”‘ Key Feature вЂ” EF DB First Approach
Unlike Code First where C# models create the database, in **DB First**:
1. Database and tables are created **first in SQL Server**
2. `Scaffold-DbContext` command **auto-generates** C# model classes from the existing database
3. Any schema changes are made in SQL Server first, then re-scaffolded
